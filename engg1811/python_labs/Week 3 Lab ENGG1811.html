Part A: Bearing to Angle Conversion

Bearings are angular measurements used in surveying and navigation. They are expressed in degrees clockwise from North (conventionally shown upwards). On the other hand, positive mathematical angles are counted anti-clockwise from the x-axis, and negative ones clockwise. For example, the red line on the diagrams below is at an angle of –45 degrees, or bearing 135 degrees.

To use trigonometric functions with angles expressed as bearings, we would need to convert them to mathematical angles in the range –180 to 180 (and then probably to radians of course). Although a bearing of 270 can be mapped to an angle of 180 or -180, we prefer it to be mapped to 180.

Part B: A classification problem

In this exercise, you will write a program to classify triangles into 3 types: scalene, isosceles and equilateral. The 3 angles of a scalene triangle have different values. For an isosceles triangle, exactly 2 angles have the same value. For an equilateral triangle, all angles are the same.

You can start with this program where we have done the first part for you. The first part of the program specifies 5 test cases where each test case specifies 3 variables with identifiers a, b and c. These three variables always sum up to 180. Therefore, they can be interpreted as the angles of a given triangle. After that you should use these angles to determine the type of triangle. The test cases are:

  • One test case with the angles 50, 60 and 70. The triangle is a scalene.
  • Three test cases with the angles 70, 70 and 40. These triangles are isosceles.
  • One test case with all angles at 60. It is an equilateral.

Since there are three possible outputs, one if/else will not work because that can only deal with 2 possibilities. You can use either nested-if or chain selection. A possibility is to use a piece code like the following and you need to fill in the logical conditions. The classification does not have to follow the order in this example, there are many possibilities.

if            :

    print('Equilateral triangle')

else:

    if                  :

        print('Isosceles triangle')

    else:

        print('Scalene triangle')

Don't forget that incremental development will be useful here. To test the program, you need to use all the five test cases in the given starting code one by one. You will need to make sure that only one test case has not been commented out.

Part C: Earth's atmosphere

The Earth's atmosphere can be divided into 5 primary layers, where each layer covers an altitude region. Instead of using 5 layers, we will use 4 layers in this exercise.

Layer Altitude region Notes
Troposphere 0-17km including 17km
Stratosphere 17-60km excluding 17km, including 60km
Mesosphere 60-120 km excluding 60km, including 120km
Thermo- and Exosphere > 120km

You are asked to write a program which takes an altitude input from the user and determine which layer of the atmosphere the altitude belongs to.

You can use the example on classifying a mark into a UNSW grade (Slides 41-45 of Week 2's lecture) that we discussed in the lecture as a guide. You should also try to make sure that your code is not redundant, see Slide 45.

Note that there are number of constants in this problem, e.g. 17, 60 etc. You should define constants at the beginning of the program and use them in your program. You should try to choose informative names for these constants.

You should think about how you should test your program and carry that out. Your tutor will ask you.
